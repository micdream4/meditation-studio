# Meditation Studio

Meditation Studio is a Next.js app for generating guided meditation scripts and audio with OpenRouter, ElevenLabs, Supabase, and Creem.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
